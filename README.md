🚀 Sistema de Gestão de Lojas e Produtos (Java)
Repositório desenvolvido para consolidar conceitos fundamentais de Lógica de Programação e Orientação a Objetos (POO) em Java. O projeto simula o gerenciamento de informações de lojas, endereços e controle de validade de produtos.
🛠️ Tecnologias e Conceitos
•	Linguagem: Java
•	Paradigma: Orientação a Objetos
•	Conceitos Aplicados:
o	Encapsulamento (Atributos privados e métodos Getters/Setters).
o	Composição de objetos (Lojas e Produtos utilizam classes de Data e Endereço).
o	Lógica de validação (Algoritmo para cálculo de ano bissexto e verificação de datas de validade).
o	Manipulação de entrada/saída via console.
📂 Estrutura do Projeto
•	Data.java: Gerencia datas com validação de dias, meses e anos bissextos.
•	Endereco.java: Estrutura de dados para localização das lojas.
•	Loja.java: Entidade principal com métodos para cálculo de gastos salariais e classificação de porte da empresa.
•	Produto.java: Gerencia informações de produtos e contém a lógica de verificação de vencimento (estaVencido).
•	Main.java: Ponto de entrada do sistema com um menu interativo.
📋 Funcionalidades
•	Cadastro de lojas com endereço completo.
•	Cadastro de produtos com data de validade.
•	Verificação automática de validade comparando com a data atual.
•	Classificação de porte da loja baseado no número de funcionários.
•	Cálculo de folha salarial.
🚀 Como executar
1.	Clone o repositório:
bash
git clone https://github.com

2.	Compile os arquivos:
bash
javac *.java
3.	Execute o programa:
bash
java Main
